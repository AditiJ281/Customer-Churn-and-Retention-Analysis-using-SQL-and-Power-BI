
# Customer Churn & Retention Analysis Dashboard

## Project Overview
This project presents an end-to-end data analysis workflow examining customer churn and retention within a banking dataset. The objective is to identify high-risk customer segments, analyze revenue exposure, and uncover factors influencing long-term retention.

The solution integrates SQL-based data preparation with Power BI visualization to deliver structured, business-ready insights.

---

## 🔄 End-to-End Workflow

### 1. Data Extraction & Transformation (SQL)
- Queried raw customer data  
- Cleaned and handled missing values  
- Created churn flags and calculated fields  
- Segmented customers into age groups, tenure bands, and balance ranges  
- Built analytical views for reporting  

Prepared datasets were structured specifically for BI consumption.

### 2.Data Modeling & Visualization (Power BI)
- Imported transformed SQL views  
- Created DAX measures for churn rate and retention rate  
- Designed KPI-driven dashboard layout  
- Built interactive filtering across geography, age, tenure, balance, and product holding  

---

## 📊 Dashboard Structure

### Churn Overview
- Churn rate analysis

### Retention Analysis
